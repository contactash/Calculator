**Simple Calculator**


It can add, subtract, divide, multiply and reset the calculator.

Getting started
`npm install`

To start the application in browser
`npm start`

To run unit tests
`npm test`

To run end to end test
`npm run cypress:chrome` or `npm run cypress:open`
